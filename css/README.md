# UkrMova Pro

Гра для вивчення української мови.

### Що вміє гра:
- Реєстрація та вхід
- Відповідати на питання і заробляти гроші
- Купувати предмети в магазині (золото, корона, VIP тощо)
- Теми: Іменники, Прикметники, Числівники, Фразеологізми
- Адмін-панель (натисни 5 разів на "МЕНЮ")
- Таблиця лідерів

### Як запустити:
Просто відкрий файл **index.html** у браузері.

### Структура папок:
- index.html
- css/style.css
- js/main.js
- js/themes.js
- assets/images/diagram-architecture.png

Зроблено для навчання української мови у цікавій формі.
